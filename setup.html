⚙️ Airtable 配置向导

📌 第一步：创建 Airtable Base

1. 登录 airtable.com
2. 创建一个新的 Base，命名为"LINN 面部歪斜评估系统"
3. 在 Base 中创建以下4张表（字段名必须完全一致）：
表名关键字段
AssessorsName(文本), Email(文本), Password(文本)
ClientsName(文本), Gender(单选), Age(数字), Assessor(文本), Phone(电话)
AssessmentsClient(关联), Assessor(文本), Status(单选), PseudoScore(数字), TrueScore(数字), ResultType(单选), CreatedAt(日期)
RecommendationsType(单选), SubType(文本), Title(文本), Content(长文本), Priority(数字)

🔑 第二步：填写配置信息

✅ 配置已保存！可以开始使用了。
在 airtable.com/create/tokens 创建，勾选 data.records:read 和 data.records:write 权限
Base URL 中的 app 后面那一段，如 appXXXXXXXXXXXXXX

📋 第三步：验证与开始

配置保存后，返回登录页即可使用系统。
演示模式无需注册，可直接体验完整流程。
