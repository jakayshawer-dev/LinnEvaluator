# LINN 面部歪斜评估系统 商用版 — 完整规格说明书

## 1. 项目概述

- **项目名称**：LINN 面部歪斜评估系统 专业版
- **项目定位**：面向内部评估师的真假大小脸商用评估系统
- **配色方案**：绿色系 + 紫色系（专业风格）
- **联系方式**：WeChat: LinnChang2017 / ins: Linn_Aesthetic
- **数据存储**：Airtable（免费层）

---

## 2. 页面流程

```
登录页 → 客户列表 → 新建评估 → 基础信息 → 总览预判（7题）
       → 面部偏斜基准（5项） → 假性评估（10项） → 真性评估（5型） → 综合结果 → 导出/打印
```

### 页面清单

| 页面 | 文件名 | 功能 |
|------|--------|------|
| 登录页 | `index.html` | 评估师登录/认证 |
| 客户列表 | `clients.html` | 查看所有客户，新建/搜索客户 |
| 基础信息 | `basic-info.html` | 录入客户基础信息 |
| 总览预判 | `overview.html` | 7道题快速预判真假方向（内部参考） |
| 面部偏斜基准 | `face-benchmark.html` | 5项面部基准数据录入 |
| 假性评估 | `pseudo-assessment.html` | 10项假性因素评估（分2页） |
| 真性评估 | `true-assessment.html` | 5型真性因素评估 |
| 综合结果 | `result.html` | 主次判断、占比、建议 |

### 总览预判页（Overview）规则
- 共7道题，每题可选答案
- 每题可展开"查看判定说明"折叠区（默认收起，供内部评估师参考）
- 页面底部固定提示："本页结果为总览预判，不替代最终结论。请继续完成假性评估模块与真性评估模块。"
- 右上角或底部固定显示当前面部主偏斜方向：左 / 右 / 不明确

## 4. 评分逻辑

### 4.1 假性评估（10项）

每项字段：
- `C`（是否异常）：是/否
- `D`（偏斜方向）：左/右/无/不适用
- `E`（严重度）：0=轻微、1=中度、2=明显
- `remark`：备注

**评分公式：**
- C=否 → 本项得分 = 0
- C=是，D=否 → 本项得分 = E × 0.5
- C=是，D=是 → 本项得分 = 1 + E

### 4.2 真性评估（5型）

每型字段：
- `fit_level`：否(0)/轻度符合(1)/中度符合(2)/高度符合(3)
- `direction`：左/右/不适用
- `dominant`：是否为当前主导矛盾 是/否
- `remark`：备注

**评分公式：**
- fit_level = 0 → 本型得分 = 0
- fit_level = 1 → 本型得分 = 1
- fit_level = 2 → 本型得分 = 2
- fit_level = 3 → 本型得分 = 3
- 若 dominant=是 → 再 +1

### 4.3 主次判断规则

```
假性主导型：假性得分 ≥ 真性得分 + 3
真性主导型：真性得分 ≥ 假性得分 + 3
真假混合型：|假性得分 - 真性得分| < 3，且两者都 > 0
```

### 4.4 占比计算

```
假性占比 = 假性得分 / (假性得分 + 真性得分) × 100%
真性占比 = 真性得分 / (假性得分 + 真性得分) × 100%
```
